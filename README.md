# Diabetic Retinopathy Detection

This project detects diabetic retinopathy from retinal images using a deep learning model and provides a simple Flask web interface.

## Features
- Upload retinal image
- Deep learning model predicts diabetic retinopathy
- Web interface built with Flask
- Displays prediction results in browser

## Technologies Used
- Python
- TensorFlow / Keras
- Flask
- HTML / CSS

## Project Structure
app.py – Flask application  
blindness_detection.py – Model training / prediction logic  
templates/ – HTML pages for web interface  
static/ – Uploaded images and assets  

## How to Run

1. Install dependencies

2. Run the application

3. Open browser

## Note
The training dataset is not included in this repository due to size limitations.
